dee 0.1.0 (development)
=======================

New features
------------

* `dee()` creates "dee" class objects.
* Absolute coordinate helpers:

  + `A()`
  + `AZ()`
  + `C()`
  + `CZ()`
  + `H()`
  + `HZ()`
  + `L()`
  + `LZ()`
  + `M()`
  + `MZ()`
  + `Q()`
  + `QZ()`
  + `S()`
  + `SZ()`
  + `T()`
  + `TZ()`
  + `V()`
  + `VZ()`
  + `Z()`

* Relative coordinate helpers:

  + `aa()`
  + `az()`
  + `cc()`
  + `cz()`
  + `hh()`
  + `hz()`
  + `ll()`
  + `lz()`
  + `mm()`
  + `mz()`
  + `qq()`
  + `qz()`
  + `ss()`
  + `sz()`
  + `tt()`
  + `tz()`
  + `vv()`
  + `vz()`
  + `zz()`
